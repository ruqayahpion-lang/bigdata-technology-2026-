# Laporan Modul 6: Spark DataFrame
* **Aktivitas**: Eksplorasi komputasi batch menggunakan PySpark DataFrame API dengan optimasi pembacaan data langsung di memori RAM (*in-memory*).
* **Analisis**: Spark memangkas siklus I/O harddisk tradisional milik MapReduce, menghasilkan eksekusi data warehouse hingga puluhan kali lebih cepat.
