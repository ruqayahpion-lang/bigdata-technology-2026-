# Laporan Modul 4: MapReduce
* **Aktivitas**: Pembuatan skrip `mapper.py` untuk tokenisasi kata dan `reducer.py` untuk akumulasi frekuensi kata via Hadoop Streaming Python.
* **Analisis**: Pemrosesan batch berpatokan pada pemisahan tugas baca (Map) baru kemudian digabungkan kembali dalam fase agregasi (Reduce).
