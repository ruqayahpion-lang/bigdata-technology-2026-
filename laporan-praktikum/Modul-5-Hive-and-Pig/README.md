# Laporan Modul 5: Hive & Pig
* **Aktivitas**: Pembuatan query warehousing dengan HiveQL dan manipulasi pipeline data transformatif menggunakan script Apache Pig Latin.
* **Analisis**: Hive mempermudah pengguna SQL, sementara Pig menyediakan fleksibilitas prosedural untuk kebutuhan ETL data semi-terstruktur.
