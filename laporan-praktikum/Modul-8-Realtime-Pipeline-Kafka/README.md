# Laporan Modul 8: Realtime Pipeline Kafka & Spark
* **Aktivitas**: Integrasi pengolahan data streaming real-time menggunakan broker pesan Apache Kafka, Spark Structured Streaming, dan UI Streamlit.
* **Fitur Keamanan**: Sensor digit rekening nasabah (`rekening_masked`), enkripsi nominal dengan Base64 (`jumlah_encrypted`), serta klasifikasi transaksi mencurigakan secara instan (`status` = FRAUD).
