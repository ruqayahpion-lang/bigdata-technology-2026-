# Laporan Modul 2: Linux Basics
* **Aktivitas**: Eksplorasi perintah navigasi CLI (`cd`, `ls -la`), manajemen hak akses (`chmod +x`), dan pembaruan environment via `source ~/.bashrc`.
* **Analisis**: Penguasaan shell Linux sangat krusial karena mayoritas infrastruktur data berskala besar dioperasikan dalam mode teks tanpa GUI.
